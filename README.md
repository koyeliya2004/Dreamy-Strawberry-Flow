# 🍓 Dreamy Strawberry Flow

A cinematic, scroll-based interactive website showcasing aesthetic strawberry drinks with smooth animations and modern UI design.

🔗 Live Demo: https://dreamy-strawberry-flow-strawberry-e.vercel.app/

---

## 💖 About The Project

Dreamy Strawberry Flow is a visually immersive web experience designed to combine:

* 🎬 Cinematic video backgrounds
* ✨ Smooth scroll-based animations
* 💎 Glassmorphism UI design
* 🍓 Aesthetic product storytelling

The goal of this project is to create a modern, high-end product landing page similar to premium brand websites.

---

## 🚀 Features

* 🎥 Fullscreen background video
* 🔄 Scroll-based interactive experience
* ✨ Smooth animations using modern UI techniques
* 💖 Glassmorphism cards and soft pastel design
* 🌸 Aesthetic drink menu with detailed descriptions
* 📱 Responsive design for all devices
